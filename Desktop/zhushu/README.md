# 《筑书》本地部署说明

## 快速启动

### 方法一：双击启动脚本（推荐）

1. 双击运行 `启动服务器.bat`
2. 等待服务器启动
3. 在浏览器中访问：**http://localhost:8000/zhushu.html**

## 功能特色

### 🎯 学徒游览路线
- **游览顺序**：京派 → 晋派 → 苏派 → 徽派 → 闽派 → 粤派 → 川派 → 海派
- **起点**：皇城北京
- **终点**：海派上海

### 📖 电子护照系统
- 收集八大派系印章
- 实时显示游览进度
- 称号升级系统（初学者 → 建筑大师）

### 🏛️ 展馆特色
- 每个派系有独特的颜色标识
- 详细的建筑特色介绍
- 代表建筑展示

## 访问地址

- **网站首页**：http://localhost:8000/zhushu.html
- **自动跳转页**：http://localhost:8000/

---

## 注意事项

1. **保持服务器运行**：关闭服务器脚本后，网站将无法访问
2. **端口占用**：如果8000端口被占用，可以修改 `启动服务器.bat` 中的端口号
3. **刷新页面**：修改代码后，按 F5 刷新浏览器即可看到更新
